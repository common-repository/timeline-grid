	Awesome Timeline grid for query post from any post type and display on Timeline.

== Description ==

Query post from your custom post types and create awesome facebook style Timeline Grid.

Use any where by short-codes and easy customize via short-code generator page, you can display or hide grid meta items like post authors, post date, post categories, post tags and etc.

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Timeline Grid</strong>" activate it.<br />

After activate plugin you will see "Timeline Grid" menu at left side on WordPress dashboard click "New Timeline Grid" and use the options field "Timeline Grid"<br />

<br />
<strong>How to use on page or post</strong><br />
When Timeline Grid options setup done please publish Timeline Grid as like post or page<br />

and then copy shortcode from top of <strong>Timeline Grid Options</strong> `[timeline_grid id="5048"]`<br />

then paste this shortcode anywhere in your page to display grid<br />
